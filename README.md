# JdbcTemplate
This project demonstrate how to integrate JdbcTemplate in a Spring MVC application.

The sample application is developed using the following pieces of software/tech:
+ Java 8
+ IntelliJ IDEA
+ Spring MVC framework 4.0+
+ JSTL 1.2
+ MariaDB 10.0+
+ Maven 3
